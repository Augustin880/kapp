Intro

Use Kapp by building one language at a time.

Kapp is a tool to help you along your language-learning journey, keep track of your progress, and centralize what you learn in one place. Pick a target language, then build your own decks, grammar pages, and reading texts around it.

Start Here

First setup

1. Add a target language Use the language area in the header. Everything in the app is filtered by the selected language.
2. Create a deck Open the Decks tab, create a deck, then add the first cards you want to learn.
3. Add support material Create grammar pages and import texts so your cards can link to real explanations and examples.

Daily Use

Simple study loop

Study

Use Flashcards for swipe review, Multiple choice for recognition, and Typing for active recall.

Read

Open a text in Library, read it in sections, and send useful words or sentences into a deck.

Understand

Open Grammar pages when a card or text links to a rule, then go back to study with that context.

Review progress

Use Statistics to see what is studied, what is known, and which cards still need more work.
